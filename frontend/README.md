# CarRent Frontend

React + Vite + Tailwind CSS frontend for the CarRent car rental system.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs at http://localhost:5173. API requests are proxied to the backend at http://localhost:8080.

## Build

```bash
npm run build
```
